# HTML Tables: Rows, Columns, and Advanced Attributes

This repository contains my practice work and notes from **Lecture 7** of the Web Development Course. In this lecture, I explored how HTML tables are used to organize and display data in a structured format using rows, columns, and advanced table attributes.

---

## 📚 Topics Covered

* Introduction to HTML Tables
* Table Rows (`<tr>`)
* Table Data Cells (`<td>`)
* Table Headers (`<th>`)
* Table Captions (`<caption>`)
* `rowspan` Attribute
* `colspan` Attribute
* Creating Complex Table Layouts
* Organizing Data Efficiently in Tables

## 🎯 What I Learned

Through this lecture, I learned:

* How to create tables in HTML
* The difference between table rows, headers, and data cells
* How to merge multiple rows using `rowspan`

* How to structure tabular data in a clean and readable format
* Best practices for creating organized HTML tables
